## iOS 26 LIQUID GLASS DESIGN SYSTEM

This app should follow Apple's iOS 26 Liquid Glass design language (WWDC 2025).
See `/Users/home/Documents/iOS/dev-docs/guides/design-system.md` for full details.

### Automatic Updates (just build with Xcode 26)
- Floating glass tab bars and sidebars
- Glass toolbar items with automatic grouping
- Controls animate with glass effects

### Navigation Pattern
Choose based on your app's needs:
- **TabView** - Great for iPhone-first apps with 3-5 main sections
- **NavigationSplitView** - Great for iPad/Mac-first or document-based apps
- **TabView + .sidebarAdaptable** - Universal apps that adapt to device

### Key iOS 26 Modifiers
```swift
// Glass effects
.glassEffect()                           // Apply glass to custom views
.buttonStyle(.glass)                     // Glass button style

// TabView enhancements
.tabBarMinimizeBehavior(.onScrollDown)   // Tab bar shrinks on scroll
.tabViewBottomAccessory { }              // Add view above tab bar
Tab(..., role: .search)                  // Search tab morphs into field

// NavigationSplitView enhancements
.backgroundExtensionEffect()             // Content extends under sidebar

// Toolbars
ToolbarSpacer(.fixed)                    // Group toolbar items
```

### Required Actions
1. Build with Xcode 26 (automatic glass appearance)
2. Remove opaque backgrounds blocking glass effects
3. Use `.glassEffect()` for custom floating elements
4. Update app icons with Icon Composer (multi-layer)

### Accessibility
- VoiceOver support with proper labels
- Dynamic Type support
- Minimum 44pt touch targets
- Reduced motion support
